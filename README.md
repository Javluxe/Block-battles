Block Battles 🎮
A Grid-Based Strategy Game in 8086 Assembly Language

What is Block Battles?
Block Battles is a fun two-player strategy game built in 8086 Assembly Language and played on the emu8086 emulator. It is inspired by the classic Battleship game.
You and the computer each have a 5x5 grid. You secretly place 3 blocks on your grid and the computer does the same. Then you take turns firing at each other by entering row and column numbers. The first one to destroy all 3 of the opponent's blocks wins!

How to Play:
Main Menu
When the game starts you will see the main menu with 3 options:

Press 1 to Play Game
Press 2 to read the Instructions
Press 3 to Quit

Placing Your Blocks

You will see your empty 5x5 grid
Enter a row number (1-5) and a column number (1-5) to place each block
Do this 3 times to place all your blocks
You cannot place two blocks on the same spot!

Game Screen

You will see two grids side by side
YOUR GRID — shows your blocks and where the computer fired
ENEMY GRID — shows where you fired (enemy blocks are hidden!)
Enter row and column to fire at the enemy
The computer will fire back automatically after your turn


Destroy all 3 enemy blocks first to win! 🏆
If the computer destroys all your blocks first, it's Game Over!
After each game you can Play Again, go back to Main Menu or Quit
